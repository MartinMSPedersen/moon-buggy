Moon-buggy

Moon-buggy is a simple character graphics game, where you drive some kind of car across the moon's surface. Unfortunately there are dangerous craters there. Fortunately your car can jump over them!

This web page is a temporary means to download the moon-buggy sources. There are two versions available:

  • The current stable version mbuggy-0.4.tar.gz. This file may also be found on sunsite and was announced in the newsgroup "comp.os.linux.announce".
  • The experimental version moon-buggy-0.4.0.91.tar.gz is in preparation for the future "0.4.1" release. New in this version:
    • Bug fixes
    • Removed the old "--with-setuid" configure option in favour of "--with-setgid". Refer to the manual for details.
    • Renamed the package from old "mbuggy" to "moon-buggy" again. As the info file's name has changed with this, you should remove any old moon-buggy versions before installing this one.
